Rule based G2P / micro service API

List available languages

G2Ps: /rbg2p/g2p/list
Syllabifiers: /rbg2p/syll/list

Transcribe a word (JSON)

Transcribe an input word using the specified language's g2p rules and return the result as json

URL: /rbg2p/transcribe/LANG/WORD

Examples:
/rbg2p/transcribe/ipa_test/hot
/rbg2p/transcribe/sws/hit
/rbg2p/transcribe/sws/dusch

Transcribe a word (XML)

Transcribe an input word using the specified language's g2p rules and return the result as xml

URLS: /rbg2p/transcribe/LANG/WORD?format=xml, /rbg2p/xmltranscribe/LANG/WORD

Examples:
/rbg2p/transcribe/ipa_test/hot?format=xml
/rbg2p/transcribe/sws/hit?format=xml
/rbg2p/transcribe/sws/dusch?format=xml

For legacy calls from ltool/yalt:
/rbg2p/xmltranscribe/ipa_test/hot
/rbg2p/xmltranscribe/sws/hit
/rbg2p/xmltranscribe/sws/dusch

Divide a transcription into syllables

Divide a transcription into syllables based on the specified language's syllable definition (if any)

URL: /rbg2p/syllabify/LANG/TRANS

Examples:
/rbg2p/syllabify/sws/d%20u0%20S%20a"
